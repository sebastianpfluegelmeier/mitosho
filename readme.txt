mido is a software utility script for musicians. with mido you can controll shortcuts
of e.g. your DAW with a midi controller.
you can set up your mappings in the config file. there allready are mappings. you can
delete them. their purpose is mainly to demonstrate the syntax of the mapping.
it has been tested fractional for osx but it should work for all plattforms.

dependencies:
    python3
    mido
    pyuserinput

if you have homebrew you can install python3 by typing:
>> brew update
>> brew install python3

if you have pip3 you can install mido and pyuserinput by typing:
>> pip3 update
>> pip3 install pyuserinput
>> pip3 install mido
